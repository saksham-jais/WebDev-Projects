Welcome to the **WebDev-Projects** repository! This workspace contains various web development projects and full-stack applications.

## 📂 Projects Overview

### 1. [SPIDE-PROJECT (CivicFlow)](./SPIDE-PROJECT/)
**CivicFlow** is a comprehensive Municipal Management System designed to streamline e-governance, citizen services, and municipal administration.
- **Live Demo:** [https://spide-project.vercel.app/](https://spide-project.vercel.app/)
- **Tech Stack:** Next.js (Frontend), Spring Boot (Backend), PostgreSQL (Database).
- **Key Features:** Role-based access control, E-Services portal, Surveyor & Ward management, and a secure directory for officials.

### 2. [Infotech Wizard Intern Assignment](./infotech-project/)
A pixel-perfect, fully responsive landing page developed as part of a Full Stack Intern assignment.
- **Live Demo:** [https://infotech-project-theta.vercel.app/](https://infotech-project-theta.vercel.app/)
- **Tech Stack:** React, Vite, Tailwind CSS, Radix UI.
- **Key Features:** Clean UI/UX, responsive across Desktop/Tablet/Mobile views, built from Figma designs.
